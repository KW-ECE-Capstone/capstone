## [test_Audio.py]

+ 입력: 사용자의 음성
+ 출력: 음성에 맞는 명령어
+ 설명: 사용자가 음성을 입력하면 키워드를 캐치하여 명령어로 반환한다. ex) 재생해줘 --> "재생"

## [pyserver.py]

+ 입력: 소켓통신으로 넘어온 음성 명령어
+ 출력: 명령어를 텍스트로 출력

+ 설명: [test_Audio.py]에서 생성된 명령어를 통신 가능한지 테스트해보는 파일이다. 해당 명령어가 정상적으로 넘어왔다면 터미널 창에 이를 출력한다.

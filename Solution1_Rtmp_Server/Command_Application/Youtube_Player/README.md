# 유튜브 플레이어

## 사용 방법

1. MainActivity API에 본인 API key를 입력한다.
2. videoID에 유튜브 videoID를 입력한다.

    ex)https://www.youtube.com/watch?v=abcdef

    "abcdef": 유튜브 videoID
3. app 모듈을 실행한다.
